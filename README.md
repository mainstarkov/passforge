# PassForge

Небольшой генератор паролей для терминала. Кроме самой генерации умеет
объяснять, насколько пароль вообще стойкий — считает энтропию и прикидывает,
сколько будет перебирать его брутфорсом при 10 млрд попыток в секунду.

Генерация идёт через `os.urandom`, а не `random.choice` на коленке — так что
годится не только для тестовых аккаунтов.

## Установка

Ничего ставить не надо, только Python 3.10+.

```bash
git clone https://github.com/mainstarkov/passforge.git
cd passforge
python3 passforge.py
```

## Как пользоваться

```bash
python3 passforge.py                       # пароль на 16 символов
python3 passforge.py -l 32                 # длиннее
python3 passforge.py -n 5                  # сразу пачкой
python3 passforge.py --no-symbols          # только буквы и цифры
python3 passforge.py --exclude "0Ol1"      # убрать символы, которые путаются
python3 passforge.py --analyze "qwerty123" # проверить существующий пароль
```

Вывод — энтропия в битах, размер алфавита, оценка "на сколько лет хватит" и
человеко-читаемая шкала стойкости (weak/fair/good/excellent).

## Ограничения

Оценка времени взлома — грубая прикидка (фиксированная скорость перебора),
не учитывает rainbow tables, утечки баз и прочие реальные векторы. Для "быстро
прикинуть, норм ли пароль" — хватает, для серьёзного security-аудита — нет.
